# SortingAlgorithm

## 使用JAVA实现的几大排序算法


| 排序方法      | 平均时间           | 最坏情况      | 辅助存储     |
| --------------|:------------------:| -------------:|-------------:|
| 简单排序      | O(n^2)             |  O(n^2)       |  O(n^2)      |
| 快速排序      | O(nlogn)             |  O(n^2)       |  O(logn)      |
| 堆排序        | O(nlogn)             | O(nlogn)       |  O(1)      |
| 归并排序      | O(nlogn)             |  O(nlogn)       |  O(n)      |
| 基数排序      | O(d(n+rd))             |  O(d(n+rd))       |  O(rd)      |
